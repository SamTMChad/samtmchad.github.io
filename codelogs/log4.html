Code Log 4: CSS Media Colors

What do the color, background-color, and border-color CSS properties do?

Explanation:

'color' 'background-color' 'border-color'
Defines the main/foreground color of the element, like the color of the text inside that element. Sets the fill/background of an element, or the area behind the content, padding, and border. It doesn't affect the area outside the border This sets the color of the border of an element. To make it visible, the element must have a border-style

How does CSS accept color as a value?

  • Named: By keywords (the name of the color) such as blue or pink, can include specifics too like "light" or "dark"
  • Hexadecimal: A hexadecimal color is specified with: #RRGGBB, where the RR (red), GG (green) and BB (blue)
  • RGB: specified with the rgb() function. Each parameter (red, green, and blue) defines the intensity of the color and can be an integer between 0 and 255
  • HSL: stands for hue, saturation, and lightness. Hue is a degree on the color wheel, Sat and Lightness are a percentage of intensity.
